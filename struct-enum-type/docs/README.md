# Struct and Enum Learning Docs

This folder documents the Rust examples in `src`.

The documentation mirrors the source tree under `docs/code/src`. For example:

- `src/basic_struct.rs` -> `docs/code/src/basic_struct.md`
- `src/advanced_topics/lifetimes.rs` -> `docs/code/src/advanced_topics/lifetimes.md`
- `src/std_like_internals/my_result.rs` -> `docs/code/src/std_like_internals/my_result.md`

Each file explains the scenario, the main Rust idea, and what to notice in the code.
